# Workshops

## 准备工作

### 安装 Rust

使用 Rustup 来安装 Rust 编译器

```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup update
rustup update nightly
```

### 将 WASM 编译器添加到工具链中

```shell
rustup target add wasm32-unknown-unknown --toolchain nightly
```

### 查看版本

```shell
rustup -V
rustc -V
```
